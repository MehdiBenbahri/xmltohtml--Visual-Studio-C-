---XML TO HTML ---

Ce Script fonctionne seulement pour la documentation généré par Visual Studio 
en C# je ne sais pas si il génére le même type de doc xml pour chaque type de projet (C#,C++,Winform ect..)
(normalement non)

C'est simple:
-vous installez le projet comme un projet php de base
-vous ouvrez xmltohtml.php 
-mettez le chemin du xml.

Si il y a une erreur vérifier si votre chemin est correcte et que votre Fichier suit bien la norme Visual Studio c#

Et ensuite mon script va faire le reste.

Le fichier 'oxfo.xml' est un example
